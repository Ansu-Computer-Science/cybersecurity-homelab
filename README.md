# Cybersecurity Homelab – IDS and IPS Implementation

## Overview
This project involves building a cybersecurity homelab environment to simulate real-world network security monitoring. The lab uses virtual machines to create a controlled environment where intrusion detection and prevention systems can be tested and analyzed.

## Purpose
The goal of this project is to understand how cybersecurity professionals detect and respond to network threats using security monitoring tools.

## Tools and Technologies
- Kali Linux
- Ubuntu Server
- Snort (Intrusion Detection System)
- OSSEC (Host-based Intrusion Detection System)
- Nessus Essentials (Vulnerability Scanner)
- VirtualBox

## Lab Architecture
The homelab environment consists of multiple virtual machines connected through a simulated network. This setup allows testing of attacks, monitoring logs, and analyzing security alerts.

## Skills Demonstrated
- Network security monitoring
- Intrusion detection and prevention systems
- Log analysis and threat detection
- Security tool configuration

## What I Learned
Through this lab I gained hands-on experience with cybersecurity tools used in real-world security operations centers (SOC). I learned how IDS and IPS systems detect suspicious activity and how security analysts monitor network traffic.
